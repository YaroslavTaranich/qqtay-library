Modal Windows

Open Example Modal

To create modal window whith click on some trigger from Javascript use this code:

$('#trigger').click(() => $('#trigger').createModal({
                text: {
                    title: 'Modal from JS',
                    body: 'Lorem ipsum dolor sit amet consectetur adipisicing elit. Doloribus voluptatem dolore non incidunt harum libero minus amet dicta, autem, ex odio nisi iure optio quaerat.'
                },
                btns: {
                    count: 3, // count of buttons
                    settings: [
                        [
                            'Close', // name of button
                            ['btn-danger', 'mr-10'], // classes for button
                            true    // close modal window
                        ],
                        [
                            'Action',
                            ['btn-success'],
                            false,
                            () => { // callback 
                                alert('Some Action');
                            }
                        ],
                        [
                            'One More',
                            ['btn-warning', 'ml-10'],
                            false,
                            () => {
                                alert('Another Action');
                            }
                        ]
                    ]
                }
            }));

Trigger must look like this:

<a href="#" class="btn btn-primary" id="trigger" data-toggle="modal" data-target="#Modal">Open Example Modal</a>

Carouel

To create Caousel module from Javascript first you must create block with class "carousel-wrapper"

<div id="carousel-wrapper"></div>

You can create object with settings of carousel. At now it only use content[0] as 'src' and content[1] as 'alt'

const carouselSettings = {
                content: 
                [
                    [
                        'https://static.mk.ru/upload/entities/2020/05/19/17/articles/facebookPicture/ee/37/29/b0/d778546d8aaeb744ba9d647437d2ec2a.jpg',
                        'alt text 1'
                    ],
                    [
                        'https://сезоны-года.рф/sites/default/files/images/shkolnikam/more.jpg',
                        'alt text 2'
                    ],
                    [
                        'https://img.gazeta.ru/files3/869/7947869/upload-shutterstock_192051191-pic4_zoom-1500x1500-61458.jpg',
                        'alt text 3'
                    ]
                ]
            };

And then use this code to add slider into wrapper:

$('#carousel-wrapper').createCarousel(carouselSettings);

Animations

<div class="d-flex f-space-around">
                <button class="btn btn-danger" id="out">Fade Out</button>
                <button class="btn btn-warning" id="in">Fade In</button>
                <button class="btn btn-success" id="toggle">Fade Toggle</button>
            </div>
            <div class="d-flex f-space-around">
                <div class="fade-preview in-out"></div>
                <div class="fade-preview toggle"></div>
            </div>

To create animations use this Javascript code:

$('#out').click(() => {
                $('.fade-preview').eq(0).fadeOut(500);
            });
            $('#in').click(() => {
                $('.fade-preview').eq(0).fadeIn(500);
            });
            $('#toggle').click(() => {
                $('.fade-preview').eq(1).fadeToggle(500);
            });
